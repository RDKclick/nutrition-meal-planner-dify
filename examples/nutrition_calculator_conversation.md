# 营养计算器对话示例

## 基础信息收集

用户：我想计算一下我的营养需求
助手：好的，我来帮您计算营养需求。请告诉我您的基本信息：
1. 您的年龄是多少岁？
2. 性别是？
3. 身高多少厘米？
4. 体重多少公斤？
5. 平时的运动情况如何？
6. 您的健康目标是什么？

用户：我今年28岁，男性，178cm，75kg，每周运动3次，想要增肌
助手：好的，我来为您计算详细的营养需求。

根据您提供的信息：
- 年龄：28岁
- 性别：男性
- 身高：178cm
- 体重：75kg
- 活动水平：中度活动（每周运动3-5次）
- 健康目标：增肌

1. 体格数据分析：
   - BMI = 75 ÷ (1.78)² = 23.7
   - BMI评估：正常范围（18.5-24.9）
   - 理想体重范围：67-77kg
   - 当前体重状态：适中

2. 能量需求计算：
   - 基础代谢率(BMR) = 1757卡路里
     (10 × 75 + 6.25 × 178 - 5 × 28 + 5)
   - 每日总能量消耗(TDEE) = 2723卡路里
     (1757 × 1.55[活动系数])
   - 增肌目标调整后热量 = 3023卡路里
     (TDEE + 300)

3. 营养素分配方案：
   蛋白质：
   - 每日需求：180g (体重×2.4)
   - 占总热量：24%
   - 来源建议：鸡胸肉、鱼、蛋、牛奶、豆制品

   碳水化合物：
   - 每日需求：390g
   - 占总热量：52%
   - 来源建议：全谷物、燕麦、红薯、水果

   脂肪：
   - 每日需求：84g
   - 占总热量：25%
   - 来源建议：坚果、橄榄油、鳄梨

4. 每日饮食建议：
   早餐（约750卡路里）：
   - 全麦面包 2片
   - 全蛋 2个
   - 牛奶 250ml
   - 香蕉 1根
   
   午餐（约900卡路里）：
   - 糙米/红薯 200g
   - 鸡胸肉 200g
   - 西兰花 200g
   - 橄榄油 15ml
   
   晚餐（约850卡路里）：
   - 全麦面条 150g
   - 瘦牛肉 150g
   - 混合蔬菜 200g
   
   加餐（约500卡路里）：
   - 训练前：香蕉 1根 + 燕麦 50g
   - 训练后：蛋白粉 30g + 牛奶 250ml

5. 注意事项：
   - 训练日和非训练日的碳水摄入可以适当调整
   - 保持充足饮水，每日2500-3000ml
   - 建议分5-6餐进食，保持稳定供能
   - 训练后30分钟内补充蛋白质和碳水
   - 睡眠保证7-8小时，有助于恢复和增肌
